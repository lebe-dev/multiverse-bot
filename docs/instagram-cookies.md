# Cookies для Instagram

Instagram требует аутентификацию для скачивания закрытых постов, Reels и сторис. Без cookies бот может возвращать ошибку или скачивать только публичный контент.

## Как экспортировать cookies

> Используйте **Firefox** — Chrome инвалидирует cookies сразу после экспорта.

1. Установите расширение **cookies.txt** для Firefox:
   https://addons.mozilla.org/en-US/firefox/addon/cookies-txt/

2. Войдите на `instagram.com` под своим аккаунтом

3. Нажмите на иконку расширения → **Current Site** → сохранится файл `instagram.com_cookies.txt`

4. Переименуйте файл в `instagram-cookies.txt`

5. **После экспорта не используйте этот браузер для Instagram** — иначе сессия ротируется и cookies протухнут

> Рекомендуется использовать отдельный аккаунт Instagram, а не основной — на случай если Instagram заблокирует его за автоматические запросы.

## Как передать cookies боту

**Способ 1 — через Telegram (рекомендуется):**

1. Отправьте боту команду `/add_instagram_cookies`
2. Прикрепите файл `instagram-cookies.txt` следующим сообщением

Бот сохранит файл на сервере и ответит подтверждением. Доступно только для пользователей из `ADMIN_USERS`.

**Способ 2 — вручную на сервере:**

```bash
# Скопировать файл в директорию бота
scp instagram-cookies.txt user@your-server:/path/to/bot/instagram-cookies.txt
```

Убедитесь, что переменная окружения указывает на правильный путь:

```env
INSTAGRAM_COOKIES_FILE=/path/to/instagram-cookies.txt
```

## Когда cookies протухают

Instagram периодически инвалидирует сессии. Признаки протухших cookies:
- Бот возвращает ошибку авторизации или пустой результат
- Сторис перестают приходить

В этом случае повторите экспорт и загрузите новый файл через `/add_instagram_cookies`.
